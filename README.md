## ✨ Funcionalidades
- **Autenticação segura** (login/cadastro)
- **Gerenciamento de perfil** (alterar senha, deletar conta)
- **Catálogo organizado** por categorias:
  - Peixes
  - Invertebrados
  - Corais
  - Plantas
  - Acessórios
- **Interface responsiva** com sidebar interativa
